# Task Scheduler with Scraping

## Project Resume

- Uses Clean Architecture
- Uses Web Api to Load Web Scraping process
    - Start
    - Stop
    - Pause
- Manages Task Scheduler
    - Export config
    - Import config
- Use MySql to storage
- Search Log
